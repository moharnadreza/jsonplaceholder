Introduction

It's a Persian free online REST API that you can use whenever you need some fake data.
It's great for tutorials, testing new libraries, sharing code examples, ...

All HTTP methods

All HTTP methods are supported, Include GET, POST, PUT, PATCH and DELETE.

Persian data

Returns Persian (Right to Left) fake data in JSON file format that helps Iranian developers.

Include LOVE

This project is made with love (Based on English version) and It's avaiable on GitHub.

Example

Run this code in a console or from any site:

fetch('https://jsonplaceholder.ir/todos/1')
        .then(response => response.json())
        .then(json => console.log(json))

        /* Tip: you can use all http methods for
        
 making requests :) */
